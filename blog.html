SVG Animation

01.13.2020

I use animation with every webpage or site I design. Animation adds a level of interactivity and an eye-catching quality nothing else can.

Recently I have been experimenting with SVGs as well as the animation of their properties using Javascript. SVGs are elements that are easy to manipulate since the vector itself is created using editable properties. One downside of SVGs however, is the complex character combination of the path property.
